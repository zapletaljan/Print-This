# Print This!
Webová aplikace pro zaslání souborů na 3D tisk. Na stránce je formulář k vyplnění, který jsem vytvořil přes stránku [Jotform](https://www.jotform.com/). Když uživatel formulář odešle, přijde mi oznámení na mobil a soubory, které nahrál, se odešlou do mého DropBoxu. [Web zde](https://zapletaljan.github.io/Print-This/).
